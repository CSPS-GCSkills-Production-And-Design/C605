Talon de paye dans Phénix

Où puis-je trouver mon talon de paye dans Phénix?

Pour savoir comment accéder à votre talon de paye dans Phénix, suivez les étapes ci-dessous.

N'oubliez pas que vous devez être connecté au réseau du gouvernement du Canada pour avoir accès à vos talons de paye.

Pour accéder à votre talon de paye dans Phénix:

  1. Rendez-vous sur iService.
  2. À partir des applications dans iService, cliquez sur Phénix.
  3. Vous serez redirigé ver les Applications Web de la Rémunération.
  4. Sélectionnez l’hyperlien Système de paye Phénix afin de pouvoir accéder à votre compte.

Vous serez intéressé d’apprendre que votre talon de paye est disponible deux jours avant le dépôt de la paye.

Une fois dans votre session Phénix:

  1. Cliquez sur Menu principal.
  2. Cliquez sur Libre-Service.
  3. Cliquez sur Paie et rémunération.
  4. Cliquez sur Chèques paie.
  5. À partir de la sélection, choisissez le talon que vous souhaitez consulter.
Veuillez noter que les options de vos menus déroulants peuvent varier en fonction de votre type d’accès à Phénix.
