Talon de paye dans Phénix

Revenus- Heures supplémentaires

Les paiements englobant la rémunération d’heures supplémentaires (également appelées services supplémentaires) incluront le versement des heures supplémentaires approuvées par votre gestionnaire dans Phénix. Les heures supplémentaires payées seront indiquées dans la section Revenus de votre talon de paye dans Phénix.

Le talon de paye dans Phénix confirme :

  • Le type d’heures supplémentaires
  • Le code de temps supplémentaire
  • Le nombre d’heures rémunérées
  • Le taux
  • Le montant payé

Note : Si les heures supplémentaires ont été travaillées au cours de différentes périodes de paye, elles seront combinées par type d'heures supplémentaires.

Comment calculer le taux horaire des heures supplémentaires ?

Si vous êtes un employé à temps plein travaillant 37,5 heures par semaine, calculez d’abord votre taux horaire de rémunération de base (ou de rémunération intérimaire) en divisant votre salaire annuel par 1956,6 heures.

72 905$ ÷ 1956,6 = 37,26$

Multipliez ensuite ce taux horaire par le multiplicateur (1.5, 1.75, 2.0, etc.) pour obtenir le taux horaire des heures supplémentaires :

37,26$ × 1,5 = 55,89$

37,26$ × 2,0 = 74,52$

Pour en savoir plus sur la façon de calculer votre taux horaire, veuillez consulter l’Annexe 1, disponible dans la section Resources. Pour accéder à l'annexe 1, il suffit de sélectionner l'icône « Boîte à outils », puis de choisir « Ressources ».
